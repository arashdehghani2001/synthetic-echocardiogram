# Synthetic-Echocardiogram
We adapt diffusion models to cardiac domain and generate new echocardiograms from scratch. 


In the folder ([videos](videos)), we put all samles from the survey. The original label (showing realism) is written in the name of each gif. The form can also be found in the link: ([Survey](https://forms.gle/Ey9ixRLhaVWtETgdA))
